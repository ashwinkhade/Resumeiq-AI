🚀 Resumeiq-AI

AI-Powered Resume Screening System with ATS Score, Skill Gap Analysis, Resume Parsing, and Job Description Matching.

✨ Features

- ATS Score Analysis
- Resume Parsing
- Skill Gap Detection
- Job Description Matching
- Resume Recommendations
- Flask Backend
- SQLite Database
- Machine Learning Models

🛠️ Tech Stack

Frontend

- React.js
- Tailwind CSS

Backend

- Flask
- SQLAlchemy
- JWT Authentication

Machine Learning

- Scikit-Learn
- Pandas
- NumPy

Database

- SQLite

🚀 Installation

git clone https://github.com/ashwinkhade/Resumeiq-AI.git

cd Resumeiq-AI

pip install -r requirements.txt

python run.py
